# Olympic Dataset Analysis

This repository contains an in-depth analysis of the Olympic dataset, which includes a wide range of information related to Olympic Games, participants, and various other attributes. The analysis covers multiple aspects, including distribution analysis, correlation analysis, time series analysis, categorical analysis, and sentiment analysis, each supported by distinct visualizations.

## Dataset Overview

The Olympic dataset contains the following columns:

- **Name:** The name of the athlete.
- **Sex:** The gender of the athlete.
- **Age:** The age of the athlete.
- **Height:** The height of the athlete.
- **Weight:** The weight of the athlete.
- **Team:** The team or country represented by the athlete.
- **Year:** The year of the Olympic Games.
- **Season:** The season of the Olympic Games (e.g., Summer or Winter).
- **Host_City:** The host city of the Olympic Games.
- **Host_Country:** The host country of the Olympic Games.
- **Sport:** The sport in which the athlete competed.
- **Event:** The specific event within the sport.
- **GDP_Per_Capita_Constant_LCU_Value:** The GDP per capita (constant local currency units) in the host country.
- **Representing_Host:** Indicates whether the athlete represented the host country.
- **Avg_Temp:** The average temperature during the Olympic Games.
- **Medal:** Number of Medals won by the atheletes.

## Analysis Highlights

### Distribution Analysis

The distribution analysis provides insights into various attributes like distribution of height and frequency of Height. Visualizations such as histograms, box plots, and density plots reveal patterns and outliers in the data.

### Correlation Analysis

Correlation analysis explores relationships between different variables, including the correlation between age, height, and weight. Correlation matrices and scatter plots are used to visualize these relationships.

### Time Series Analysis

Time series analysis focuses on the change in various attributes over the years. Line charts are used to illustrate  Height trends and patterns in Olympic data over time based on gender.

### Categorical Analysis

Categorical analysis delves into attributes like the average height and weight of atheletes and Medal count based on gender. Bar charts offer insights into the distribution of these categorical variables.

### Sentiment Analysis

Sentiment analysis goes beyond numeric data, examining textual data such as atheletes sentiments such as 'Positive', 'Negative' and 'Neutral'. Pie charts are used to visualize the percentage of different sentiments.

Enjoy exploring the Olympic dataset analysis and gaining insights from the diverse range of visualizations!
