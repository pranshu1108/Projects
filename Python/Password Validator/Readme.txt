Introduction:-

The Password Validator Program is a simple program designed to validate user passwords based on certain criteria.

Problem Statement:-

The task is to create a program that checks the validity of a user-provided password. The program should apply specific rules and criteria to determine whether the given password meets the required security standards.

Password Criteria: -

o Minimum Length: The password should have a minimum specified length. 
o Combination of Character Types: The password should include a mix of uppercase letters, lowercase letters, digits, and symbols.
o Blank Space : The password should not contain any blank space.
